# Online Store

A fully-featured e-commerce application built with **Spring Boot**.  
This project leverages **Spring Data JPA, Hibernate, MySQL, JWT, Caching, AOP**, and **RESTful APIs**.  
Payment integration is implemented via **ZarinPal**.
